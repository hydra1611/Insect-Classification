## Create virtual Envirnment (First time during setup)

```
python -m venv captureEnv
```

## Activate venv

```
# For Windows
.\captureEnv\Scripts\activate  
 
# For Linux 
source captureEnv/bin/activate
```

## Install dependancies

```
pip install -r requirments.txt
```

## Run Command

```
python capture.py
```
